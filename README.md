# RethinkDB Dump & Restore CLI
This utility helps dump and restore your rethinkdb database.

## Installation
Using NPM
```
  npm install -g rethinkdb_dump_restore_cli
```

You now have access to `rtkdb` globally.

## Run the utility
```
  rtkdb
```
